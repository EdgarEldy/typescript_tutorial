## A typescript language tutorial using gulp.

### Folder structure :
```
typescript-tutorial/
    ├─ dist/
    └─ src/
    |.gitignore
    |gulpfile.js
    |package.json
    |package-lock.json
    |README.md
    |tsconfig.json
```
### Tools
Webstorm 2021.3.2 or vs code
###Configurations
1. Install node.js
2. Install yarn globally : npm i -g yarn
3. Install gulp globally using yarn : yarn global add gulp-cli

### Run app
1. Open cmd 
2. Run gulp 
3. Run node dist/main.js
4. Enjoy
